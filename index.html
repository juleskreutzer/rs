Webdeveloper & CMS ontwikkeling

Sinds 2009 ben ik bezig met het ontwikkelen van websites voor kleine bedrijven in midden limburg. Door mijn intresse ben ik zeer gedreven in het ontwikkelen van moderne websites, zowel meerdere pagina's als de zogenoemde 'landing page' websites. Perfectionisme en functionaliteit speelt een grote rol voor mij bij de ontwikkeling van websites.

Bij ontwikkeling van een nieuwe website maak ik het liefst gebruik van de nieuwste technologiën om uw website altijd goed te laten fucntioneren op computer, maar ook op een mobiel apparaat zoals een smartphone of tablet. Enkele van deze technologiën zijn HTML 5 en CSS 3. Tevens gebruik ik ook Bootstrap 3 als basis opmaak van de website. Dit zorgt voor een nog betere werking op mobiele apparaten.

Nu! CMS

Nu! CMS (Content Management System) is een eigen ontwikkeld systeem om uw website te kunnen beheren. Dit CMS kunt u gratis gebruiken bij uw website die door NuJules is ontwikkeld. De verwachting is dat NU! CMS in 2014 kan worden gaan gebruikt. Verdere informatie over het NU! CMS kunt u vinden op  deze pagina.

Het ontwikkelings proces.

Afspraak

Door middel van een persoonlijke afspraak stel ik uw wensen vast en probeer ik een beeld te krijgen hoe uw perfecte website eruit moet komen te zien.

Ontwerp

Op basis van uw wensen maak ik een ontwerp voor uw website als een basis om vanuit te werken.

Verbeteringen

Na het gezamelijk aanpassen van het ontwerp, gaat er een website die voldoet aan uw wensen ontwikkeld worden.

Resultaat

Na een laatste bijeenkomts om voor de laatste keer te kijken of de website voldoet aan uw wensen is hij klaar om online gezet te worden.

HTML 5

HTML 5 is de nieuwste techniek voor het opmaken van websites. HTML 5 gebruik ik voor het opbouwen van uw website om zo tot een perfect resultaat te komen.

CSS 3

CSS 3 bevat tal van nieuwe mogelijkheden voor webdevelopers. CSS 3 zorgt er uiteindelijk voor dat uw website met de juiste toepassing van HTML 5 er gelikt uit ziet.

BOOTSTRAP 3

Bootstrap is een basis gebaseerd op CSS 3 voor uw website. Er zijn vele mogelijkheden beschikbaar en het zorgt ervoor dat uw website geschikt is voor mobiele apparaten.
